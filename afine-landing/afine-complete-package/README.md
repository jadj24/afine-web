# AFINE - Cursos de Finanzas

## 📋 Contenido del Paquete

Este paquete contiene el sitio web profesional de AFINE con todas las funcionalidades.

### 📁 Estructura
- **/dist** → Archivos para SUBIR al hosting (listos para producción)
- **/src** → Código fuente React (para modificaciones)
- **/dist/video** → Videos de presentación

### 🚀 Cómo usar

**Opción 1 - Subir directamente:**
Sube el contenido de /dist a tu hosting

**Opción 2 - Modificar:**
1. pnpm install
2. pnpm run dev (desarrollo)
3. pnpm run build (generar /dist)

### 🎬 Agregar video de YouTube
En src/App.tsx, línea ~225, cambia:
`src="https://www.youtube.com/embed/YOUR_VIDEO_ID"`
por tu ID de video de YouTube
